Cryptography algorithms for TinyOS

The aim of this project is to provide Cryptography algorithms in TinyOS 2.x. All contributions are welcomed.

Implementation done:

Trivium

The Trivium stream cipher is now implemented as a TinyOS interface for 8-bit and 16-bit microcontroller. This implementation follows the original implementation available in the eSTREAM repository. Some applications using this interface are available in the apps folder. For more information consult the cipher webpage.
For more information about this implementation, consult the source code documentation.

Download

The CVS directory can be browsed at the address.
The directory can be downloaded with the CVS command:
cvs -z3 -d:pserver:anonymous@tinyos.cvs.sourceforge.net:/cvsroot/tinyos checkout -P tinyos-2.x-contrib/crypto/

Installation

To install or remove all the interfaces provided by this directory in the TinyOS tree, the scripts install.sh and uninstall.sh can be used. They must have the execution permission with the command:
chmod +x install.sh uninstall.ch
Then the command:
./install.sh
copies all the files in the TinyOS tree.

For each application example in the folder apps, there is a Makefile which can compile the application the same way as standard TinyOS applications.
