NXTMOTE Tutorials
Last updated 16 May 2007
The nxtmote is TinyOS mote on LEGO MINDSTORMS NXT. This page is an overview of the tutorials that are available for this platform.

Lesson A: Setting up the Toolchain

Lesson A shows how to set up the toolchain. It includes compiling the toolchain using RPM spec files and installing the RPM files.

Lesson B: Hello World

Lesson B contains the steps for testing the basic nxtmote with a multimeter. It drives a selected pin on one of the input ports using the Blink example.

Please note that this is TinyOS contribution project. Further information is available by writing nxtmote@gmail.com
