TinyOS Programming

This directory contains the example applications from the "TinyOS Programming" book, published by Cambridge University Press, ISBN 978-0521896061. The applications are developed for micaz motes with an mts310 sensor board, most of them should be easily ported to other motes and/or sensor boards.

  • AntiTheftBook: the successive "anti-theft" applications developed in Chapter 6.1-6.5.
  • FlashSampler: the "sample-to-flash" application developed in Chapter 6.5.3.
  • ReliableSerial: the simple reliable transmission protocol developed in Chapter 7.5.
  • SoundLocalizer : the "closest-to-sound" application developed in Chapter 13.
